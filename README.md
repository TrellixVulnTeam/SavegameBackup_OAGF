# SavegameBackup
A software for backing up save files of computer games. 

## Prerequisites

- Install [Python](https://www.python.org/)
- Install [pip](https://pip.pypa.io/)
- Install [Virtualenv](https://virtualenv.pypa.io/) with pip:
```
pip install virtualenv
